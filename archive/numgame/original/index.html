numgame

What is this?

This is an unfun game that I wrote in an hour.

Here's how it works:

Set the max value for numbers and the completion time The game creates a mathematical equation that looks like this 

%num1% %operator% %num2%
num1 and num2 are randomly generated numbers that go from 1 to the max value you gave. The operator is randomly selected, due to floating reasons the only operators that are generated now are +, - and *. You are given the amount of time you set to solve the generated equation
